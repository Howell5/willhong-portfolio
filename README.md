# willhong.dev

Personal portfolio website built with Astro, React, and a 3D Box Carousel.

## Tech Stack

- [Astro](https://astro.build) - Static site framework
- [React](https://react.dev) - Interactive components
- [TailwindCSS](https://tailwindcss.com) - Styling
- [Framer Motion](https://motion.dev) - Animations
- [Box Carousel](https://fancycomponents.dev) - 3D carousel component

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```
